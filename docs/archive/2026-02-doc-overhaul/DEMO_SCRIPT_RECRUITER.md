# Demo Script (Recruiter / Non-Technical)

_Duration options: 10 minutes / 30 minutes_

## 10-Minute Path

1. Problem: small and mid-size retailers often operate inventory manually.
2. Solution: ShelfOps predicts demand, surfaces risks, and supports reorder decisions.
3. Trust: current status is pre-production hardening with transparent readiness labels.
4. Differentiator: enterprise integration logic is built and pilot-validated without overclaiming full rollout readiness.
5. Outcome: demonstrates business judgment + technical execution across data, ML, and product delivery.

## 30-Minute Path

1. Business context and why SMB-first is intentional.
2. What is live vs pilot-validated vs blocked (readiness board walk-through).
3. How model effectiveness is measured in production.
4. How customer onboarding handles different schemas.
5. How this maps to Senior DS/MLE hiring signal.

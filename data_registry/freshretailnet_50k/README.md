# FreshRetailNet-50K

Role: secondary public benchmark for stockout-aware and censored-demand work.

Use this dataset to prove:

- stockout-aware evaluation
- censored-demand methodology
- perishable and promo-sensitive analysis
- weather-linked demand context where supported

Do not use it to claim:

- measured merchant business impact
- complete U.S. retail pilot evidence
- full end-to-end replenishment workflow proof by itself

Current repo evidence:

- raw files under `data/benchmarks/freshretailnet_50k/raw`
- benchmark report under `backend/reports/freshretailnet_stockout_benchmark.json`
- public appendix in `MODEL_CARD_STOCKOUT_APPENDIX.md`
